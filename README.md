# Java Library System - PUCPR

Academic project for the Object-Oriented Programming (OOP) course at PUCPR.

## Description
A simple library management tool to practice classes, attributes, and basic logic in Java.

## Files
- `Main.java`: Runs the application.
- `Livro.java`: Book class with attributes and methods.

## How to run
Make sure you have JDK 17+ installed.

   javac *.java
